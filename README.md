# PC System Integration Lab

Hands-on hardware build, troubleshooting, and system restoration documentation.

---

## What I Did

- Installed DDR5 RAM and configured BIOS/UEFI settings to optimize system speed and stability
- Diagnosed and resolved POST errors and "HARD DRIVE NOT FOUND" faults via CMOS replacement and SATA reseating
- Performed clean Windows installation using ISO imaging and Diskpart for advanced partitioning
- Restored motherboard functionality by reapplying thermal paste and cleaning circuitry
- Revived unresponsive hardware by troubleshooting firmware conflicts and repairing physical connection failures
